Əsaslar

ABA nədir və necə işləyir?

ABA insan davranışını elmi şəkildə öyrənən, faydalı bacarıqları öyrətmək və problemli davranışları azaltmaq üçün istifadə olunan, sübuta əsaslanan yanaşmadır.

A → B → C modeli

Antesedent (əvvəlki stimul) → Davranış → Nəticə. Nəticə davranışın təkrarlanıb-təkrarlanmayacağını müəyyən edir.

Məlumata əsaslanır

Hər qərar “hiss” ilə yox, toplanan obyektiv məlumatla verilir. İrəliləyiş rəqəmlərlə ölçülür və plan ona uyğun dəyişir.

Kimə kömək edir?

Autizm spektri, inkişaf gecikmələri, diqqət və davranış problemləri olan uşaqlara — danışıq, sosial və özünə xidmət sahələrində.
